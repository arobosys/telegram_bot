Установка зависимостей:

pip3 install schedule 
pip3 install pytelegrambotapi

Запуск:
python3 ./bot.py

Файл subscriptions содержит id подписчиков бота. 

Команды:

Для покера:
!poker
!sum

Для подписок боту в ЛС отправляется:
!subscribe
!unsubscribe

Также бот реагирует на упоминания тасков, и скидывает URL в жире на упомянутые таски.
